# Data Analytics Practice Projects

This repository contains a collection of business and financial datasets used for hands-on data analytics practice.

The aim of these exercises is to develop practical skills in:
- Data cleaning and preprocessing
- Exploratory data analysis
- KPI analysis
- Business insights extraction
- Dashboard creation

## Datasets Included

- Order Data
- Customer & Sales Data
- Financial Sample
- Loan Data
- Sales Revenue Data
- Supermarket Sales
- Cookie Data

Each folder contains the dataset and brief analysis notes.

## Tools Used
- Microsoft Excel
- Power BI
- Python (pandas, matplotlib)
- SQL (where applicable)

This repository demonstrates structured analytical thinking and practical data handling skills.
